## 🚀 Why This Exists

Studying for **CompTIA Security+** can feel overwhelming when you're only reading or watching videos.

This project turns studying into **interactive practice inside the terminal**:

* 🧠 Reinforces key concepts through repetition
* ⏱️ Simulates exam pressure with timed quizzes
* 📚 Helps memorize acronyms, ports, and controls
* 📊 Tracks study progress locally

Instead of passive learning, this tool encourages **active recall** — one of the fastest ways to retain information.

---

## ⚡ Features

✔ Domain-based quizzes (Security+ domains)
✔ Timed quiz mode for exam simulation
✔ Flashcard learning mode
✔ Missed question review system
✔ Quiz history tracking
✔ Study streak tracking
✔ Memory sheet for quick review
✔ Runs entirely in the terminal

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/security-plus-terminal-trainer.git
cd security-plus-terminal-trainer
```

Run the program:

```bash
python3 security_plus_study_app_v3.py
```

No external dependencies required.
